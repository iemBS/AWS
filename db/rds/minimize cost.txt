Minimize Cost
Note:
  -What contributes to cost
    -instance hours
      -base on region, instance type, DB engine, and license
    -DB storage
    -Backup storage
      -size of backups stored in AWS
      -no charge for backup storage up to 100% of total database storage
    -Data transfer
      -outgoing traffic only
      -regional data transfer pricing
      -includes copying region to region 
  -Tagging can escpecially help in companies with multiple systems or teams 
Main Success Scenario:
  1. Use AWS Cost Explorer
  2. Use tags to get a better breakdown
  3. Reserve instances if you can
